    Scaffolder

    Class to create scaffolders when creating folders, and copying files/templates

    Example:

    import {Scaffolder} from './utils';
const scaffolder = new Scaffolder();
await scaffolder
    .target('/path/to/copy/files')
    .copy('foo.js')
    .copy('bar.js')
    .commit();
